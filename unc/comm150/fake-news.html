Birds Aren't Real

About this Project

This page was created for COMM-150 at UNC. This site is created within these criteria - "[b]uild a site which uses outside links as an integral part, construct a faux documentary." This site is intended to serve as a faux documentary about how Birds aren't real, but robots that are part of government surveillance. Outside sources are drawn from news articles, videos and social media accounts. Upon closer inspection, some sources refute "birds aren't real", but quotes or images from these articles were able to be picked in a way that makes it seem as though the sources support the claims of the site.

What is the "Birds Aren't Real" Movement?
